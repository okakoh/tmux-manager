# Contributing

Thanks for considering a contribution.

Useful contributions include bug reports, feature requests, config examples,
documentation fixes, tests, and small implementation improvements.

When opening an issue, please include:

- OS and version
- tmux version
- Go version, if building from source
- `tmux-manager` version or commit
- a minimal config snippet, if relevant
- expected behavior and actual behavior

Japanese issues are welcome. / 日本語での Issue も歓迎です。

Please avoid posting private project paths, API keys, tokens, customer names,
or internal command arguments in public issues.
